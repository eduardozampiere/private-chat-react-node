# Chat com sistema de usuarios
* Desenvolvido com ReactJs E NodeJs.
* Banco de dados em MongoDB
* Usando Socket.io

## Features
* Chats privados
* Adicionar amigos
* Alerta quando amigo entrar no chat
* O Usuario pode ter varias conversas simultaneas
